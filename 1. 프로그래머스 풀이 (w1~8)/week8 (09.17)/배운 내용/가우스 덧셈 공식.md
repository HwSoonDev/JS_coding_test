## 가우스 덧셈 공식

![[Pasted image 20250917152238.png]]
```
총합 = (처음 수 + 끝 수) x 숫자 개수 / 2

1) total = (start + end) x num / 2 
2) end = start + num

(치환)

total 
= (2 x start + num) x num / 2

start = (2 x total / num  - num) / 2   
```
