대문자를 소문자로
소문자를 대문자로 바꾸는 문제
```js
function solution(my_string) {
    return [...my_string].map((c) => c===c.toUpperCase() ? c.toLowerCase() : c.toUpperCase()).join('');
}
```

1) toUpperCase(), toLowerCase()
   
   항상 헷갈리는 메서드.
   이게 문자열 메서드로 있었는지, 괄호 안에 문자열을 넣는지
   이름도 애매하게 기억나는 함수입니다.
   언어 별로도 달라서 더 힘듭니다.
   
2)  `c===c.toUpperCase()` 대문자인지 판단
