숫자를 문자로 변환? 하는 문제
알파벳의 아스키 코드 숫자를 자바스크립트에서 어떻게 이용하는지 배울 수 있었습니다.

```js
function solution(age) {
    return [...age.toString()]
	    .map((n)=>String.fromCharCode(97+Number(n)))
	    .join('');
}
```
