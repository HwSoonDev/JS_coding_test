
```js
function solution(cipher, code) {
    return [...cipher].filter((e, index)=> (index+1)%code===0).join('');
}
```

filter의 callback을 공부할 수 있었습니다.

#### filter()의 콜백
```js
filter((element, index, array) => /*조건*/ );
```
- `element`: 현재 요소
- `index`: 현재 요소의 인덱스
- `array`: 원본 배열

이번 문제는 인덱스로 조건 처리를 하는 게 좋아보였습니다.
