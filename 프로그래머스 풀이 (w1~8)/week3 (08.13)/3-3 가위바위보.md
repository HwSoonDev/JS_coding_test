주어진 상대방 패에
이기는 경우를 만드는 문제

```JS
function solution(rsp) {
    const winMatch = {
        '0' : '5',
        '2' : '0',
        '5' : '2',
    }
    return [...rsp].map((n)=>winMatch[n]).join('');
}
```

매칭이 되면 객체를 만들면 된다.

다른 언어에서는 Dictionary를 만들고
배열도 for문을 돌리면서 값을 만든다고 골머리를 썩혔어야 했을 텐데,
(그 정도 되면 딕셔너리 만들기로 접근 안 했을 것 같지만)

JS는 **객체**라고 하는 **마법의 지팡이**가 있어서
응 딸깍으로 만들어 버렸다.

솔직히 아직도 객체가 뭐하는 존재인지 잘 모르겠다.
