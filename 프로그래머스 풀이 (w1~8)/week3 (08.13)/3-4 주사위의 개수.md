박스 의 가로, 세로, 높이가 주어지고
주사위가 몇 개 들어하는지

내 풀이
```JS
function solution(box, n) {
    return Math.floor(box[0]/n) * Math.floor(box[1]/n) * Math.floor(box[2]/n);
}
```

다른 풀이
```JS
function solution(box, n) {
    return box.reduce((acc,v) => acc * Math.floor(v / n), 1);
}
```

확실히 reduce를 쓰면 간결해진다.

**reduce**
acc : 매 방문마다 연산 리턴 값을 저장하는 곳
v : 방문하는 요소 값
