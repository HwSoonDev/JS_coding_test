function solution(my_string) {

## 컨테이너

| 분류        | 이름                    | 설명                     |
| --------- | --------------------- | ---------------------- |
| ✅ 배열형     | `Array`               | 순서 있는 값들의 리스트          |
| ✅ 객체형     | `Object`              | 키-값 쌍으로 구성된 구조         |
| ✅ 집합형     | `Set`                 | 중복 없는 값들의 집합           |
| ✅ 맵형      | `Map`                 | 키-값 쌍 (key는 어떤 타입도 가능) |
| 📦 약한 참조형 | `WeakMap` / `WeakSet` | 가비지 컬렉션 고려한 특수한 컨테이너   |
