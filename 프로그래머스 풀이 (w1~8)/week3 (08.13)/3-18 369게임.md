```js
function solution(order) {
    return [...order.toString()]
	    .filter((e)=>e!=='0'?Number(e)%3===0:false)
	    .length;
}
```

이렇게 쓰신 분이 있었는데
```js
function solution(order) {
    var answer = [...order.toString().matchAll(/[3|6|9]/g)].length;
    return answer;
}
```
