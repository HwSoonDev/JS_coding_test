###### 문제 설명

정수 배열 `numbers`가 매개변수로 주어집니다. `numbers`의 각 원소에 두배한 원소를 가진 배열을 return하도록 solution 함수를 완성해주세요.

내 풀이
```JS
function solution(numbers) {
    return numbers.map((n) => n*2);
}
```
